🏥 Healthcare Analytics & Hospital Performance Dashboard

📌 Project Overview

The Healthcare Analytics & Hospital Performance Dashboard is an end-to-end Data Analytics project designed to analyze hospital operations, patient demographics, doctor performance, treatment insights, and revenue trends.
The project demonstrates the complete analytics workflow, including database management, SQL analysis, data modeling, and interactive dashboard development using Power BI.

---

🎯 Business Objective

The objective of this project is to transform raw healthcare data into actionable insights that can help hospital management:

- Monitor patient activity and demographics
- Analyze doctor performance and specialization trends
- Track appointments and treatment patterns
- Evaluate revenue and billing performance
- Support data-driven decision-making

---

🛠️ Tools & Technologies

- MySQL
- SQL
- Power BI
- DAX
- CSV Datasets

---

📂 Dataset Information

The project uses five interconnected healthcare datasets:

Patients

Contains patient demographic and insurance information.

Doctors

Contains doctor details, specialization, and experience information.

Appointments

Stores appointment records between patients and doctors.

Treatments

Contains treatment details and associated costs.

Billing

Stores billing, payment method, and payment status information.

---

🗄️ Database Design

The healthcare database was created in MySQL and consists of the following tables:

- Patients
- Doctors
- Appointments
- Treatments
- Billing

Relationships were established between tables to support business analysis and reporting.
